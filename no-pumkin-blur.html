Overview

Removes the pumpkin blur overlay in Bedrock Edition.

Features

  • Minecraft Bedrock compatible
  • Optimized gameplay experience
  • Easy installation and usage

Compatibility

  • Minecraft Bedrock 1.21+
  • Works in multiplayer
  • Supports Realms

Known Issues

  • May conflict with other addons editing the same features
  • Some versions may require experimental features

Tips

  • Create backups before updating worlds
  • Use with fresh worlds for best results
  • Enable required experiments

Texture Changes

  • Removes pumpkin overlay
  • Cleaner survival gameplay
  • No gameplay changes

FAQ

Does this work on Realms?

Yes, as long as addons are enabled.

Can I use this with other addons?

Usually yes, but addons editing the same systems may conflict.
