❓ FAQ

Are all tools free to use? +
Yes! Every tool on this hub is completely free and open source. No accounts, no paywalls.
Do tools work on iOS? +
The Bedrock Launcher targets Android only. Browser tools like Addon Studio and World Analyzer work on any device including iOS.
Can I contribute to the projects? +
Absolutely! All projects are open-source on GitHub. Open issues, submit PRs, or suggest features.
What Minecraft Bedrock version is supported? +
Tools are designed for Bedrock 1.20+. Older versions may work but are not officially supported.
How do I report a bug? +
Open an issue on the relevant GitHub repo. Include your device, OS version, and steps to reproduce.
